Proposal

Problem Statement

The goal of our project is to create a model, using supervised classification, that is able to identify and diagnose _______ (cardiomegaly?) from an inputted Chest X-ray. Using our system, someone can input a Chest X-ray of a patient, and our model will predict whether or not the patient has ________ (cardiomegaly?). If the X-ray diagnose is positive, the Chest X-ray will be outputted with a rectangle encasing area of interest (the enlarged heart).


Approach

We will be utilizing supervised learning to detect for features indicative of [our specified diagnosis]. This will be done by training a convolutional neural network to extract features and classify the input X-Ray images, with a certain confidence, into the various classes representing each condition. There will be some overlap between classes due to the nature of the dataset used, but we will choose the class with the highest confidence for a given image with overlapping ground truth.


Experiments and Results
